# DGSite: A Well-Calibrated Deep Learning Framework for Protein–RNA Binding Site Prediction

**DGSite: 一个用于蛋白质-RNA结合位点预测的校准良好的深度学习框架**

## 精华总结
本研究开发了DGSite深度学习框架，用于精准预测蛋白质-RNA结合位点，通过良好的模型标定提高了预测的可靠性和实用性。该框架在蛋白质-RNA相互作用研究中具有重要应用价值。

**关键词**: 深度学习、蛋白质-RNA结合位点、预测模型、模型标定
