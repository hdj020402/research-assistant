# PeptideNet: An Integrative Deep Learning Framework for Predicting Diverse Bioactive Peptides Using Protein Language Model Embeddings

**PeptideNet：利用蛋白质语言模型嵌入预测多样生物活性肽的集成深度学习框架**

## 精华总结
PeptideNet框架创新性地整合了蛋白质语言模型和深度学习技术，实现了对多类生物活性肽的高效准确预测，为肽类药物设计提供了有力工具。

**关键词**: 深度学习、蛋白质语言模型、生物活性肽、药物发现、序列预测

## 摘要（英文）
Hamza Zahid, Hilal Tayara, et al. doi.org/10.1021/acs.jcim.5c02885 Publication Date (Web):February 24, 2026 Read Article

## 摘要（中文）
本研究提出PeptideNet框架，将蛋白质语言模型嵌入与深度学习相结合，用于预测多种生物活性肽。该方法利用预训练的蛋白质语言模型捕捉肽序列的语义特征，通过集成学习策略提高预测性能和泛化能力，对药物发现和肽工程具有重要意义。
