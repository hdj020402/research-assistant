# DGSite: A Well-Calibrated Deep Learning Framework for Protein–RNA Binding Site Prediction

**DGSite：用于蛋白质-RNA结合位点预测的经过充分校准的深度学习框架**

## 精华总结
本研究开发了DGSite深度学习框架，用于高精度预测蛋白质-RNA结合位点，通过充分的模型校准确保预测置信度的可靠性。该方法整合了多种生物化学特征，在多个基准数据集上显示出优于现有方法的性能。

**关键词**: 蛋白质-RNA结合、深度学习、结合位点预测、图神经网络、模型校准

## 摘要（英文）
Ximin Zeng, Shaoping Shi, et al. doi.org/10.1021/acsomega.5c12723 Publication Date (Web):February 23, 2026 Read Article

## 摘要（中文）
蛋白质-RNA相互作用在生物学过程中起着至关重要的作用。准确预测蛋白质-RNA结合位点对于理解分子机制和药物发现至关重要。本研究提出了DGSite，一个经过充分校准的深度学习框架，用于预测蛋白质表面上的RNA结合位点。该框架采用图神经网络和深度学习方法，通过整合多种生物化学特征，包括序列信息、结构特征和进化信息，实现了高精度的结合位点预测。为了确保模型的可靠性，研究团队对预测结果进行了充分的校准，使得预测的置信度与实际的预测精度相匹配。在多个基准数据集上的验证表明，DGSite框架相比现有方法显著提高了预测精度和泛化能力。该研究为蛋白质-RNA相互作用研究提供了有力的计算工具。
