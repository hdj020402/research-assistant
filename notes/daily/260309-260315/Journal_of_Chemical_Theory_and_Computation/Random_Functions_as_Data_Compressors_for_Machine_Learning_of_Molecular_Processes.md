# Random Functions as Data Compressors for Machine Learning of Molecular Processes

**随机函数作为分子过程机器学习的数据压缩器**

## 精华总结
提出随机函数作为新颖的数据压缩策略，显著提高了分子过程机器学习模型的计算效率和预测准确性。该方法为处理复杂分子系统提供了一种有效的特征工程方法。

**关键词**: 随机函数、数据压缩、机器学习、分子过程、特征表示

## 摘要（中文）
本研究提出利用随机函数作为数据压缩方法，用于分子过程的机器学习建模。通过随机函数表示，可以有效降低高维分子数据的维度，提高机器学习模型的训练效率和泛化能力。该方法在分子动力学模拟和分子性质预测中具有潜在应用价值。
