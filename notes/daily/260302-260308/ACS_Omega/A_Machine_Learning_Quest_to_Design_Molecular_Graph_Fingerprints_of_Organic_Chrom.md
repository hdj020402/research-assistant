# A Machine Learning Quest to Design Molecular Graph Fingerprints of Organic Chromophores for Adjusting Photoluminescent Quantum Yields

**基于机器学习的有机发色团分子图指纹设计研究：用于调控光致发光量子产率**

## 精华总结
本研究利用机器学习方法设计有机发光体的分子图指纹，通过建立分子结构与光致发光量子产率之间的定量关系，实现了对有机发光材料光学性质的高效预测和优化。

**关键词**: 机器学习、分子图指纹、有机发色团、光致发光量子产率、分子设计
