# DGSite: A Well-Calibrated Deep Learning Framework for Protein–RNA Binding Site Prediction

**DGSite: 用于蛋白质-RNA结合位点预测的校准良好的深度学习框架**

## 精华总结
提出了一个校准良好的深度学习框架DGSite,能够准确预测蛋白质-RNA结合位点,为生物分子相互作用研究提供了新的计算方法。

**关键词**: 蛋白质-RNA结合、深度学习、位点预测、神经网络、生物信息学

## 摘要（英文）
Ximin Zeng, Shaoping Shi, et al. doi.org/10.1021/acsomega.5c12723 Publication Date (Web):February 23, 2026 Read Article

## 摘要（中文）
本研究开发了DGSite框架,利用深度学习技术预测蛋白质-RNA相互作用的结合位点。该方法通过构建经过良好校准的神经网络模型,实现了对结合位点的高精度识别,为蛋白质-RNA相互作用研究提供了重要工具。
