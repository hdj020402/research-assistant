# DGSite: A Well-Calibrated Deep Learning Framework for Protein–RNA Binding Site Prediction

## 摘要（英文）
Ximin Zeng, Shaoping Shi, et al. doi.org/10.1021/acsomega.5c12723 Publication Date (Web):February 23, 2026 Read Article

## 摘要（中文）
本研究提出DGSite框架,这是一个经过良好校准的深度学习模型,用于预测蛋白质-RNA结合位点。该方法结合了深度学习技术与生物信息学,能够准确识别蛋白质表面与RNA相互作用的关键位置。通过引入校准机制,提高了模型预测的可靠性和准确性,在蛋白质-RNA相互作用研究中具有重要应用价值。
