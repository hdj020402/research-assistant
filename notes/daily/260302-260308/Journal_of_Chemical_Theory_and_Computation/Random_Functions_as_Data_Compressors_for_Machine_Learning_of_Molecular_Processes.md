# Random Functions as Data Compressors for Machine Learning of Molecular Processes

**随机函数作为分子过程机器学习的数据压缩器**

## 精华总结
提出了一种基于随机函数的新颖数据压缩方法，用于加速分子过程的机器学习建模。该方法显著提高了模型训练效率和预测精度，为计算化学领域的AI应用开辟了新方向。

**关键词**: 随机函数、数据压缩、机器学习、分子建模、维度降低
