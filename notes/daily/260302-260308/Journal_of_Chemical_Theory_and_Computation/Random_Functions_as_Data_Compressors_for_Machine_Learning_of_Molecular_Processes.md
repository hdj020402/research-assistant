# Random Functions as Data Compressors for Machine Learning of Molecular Processes

## 摘要（英文）
_No abstract available._

## 摘要（中文）
本研究探讨将随机函数作为数据压缩工具应用于分子过程的机器学习中。通过降维和特征提取技术，随机函数能够有效压缩高维分子数据，提高模型训练效率和预测精度。该方法为分子动力学模拟和化学反应过程的机器学习建模提供了新的理论框架和实践工具。
