# Machine Learning Methods for Protein–Protein Interaction Prediction Based on Noncovalent Interactions

**基于非共价相互作用的蛋白质-蛋白质相互作用预测的机器学习方法**

## 精华总结
本研究开发了基于非共价相互作用的机器学习方法来预测蛋白质-蛋白质相互作用，通过计算分子间的范德华力、静电相互作用和氢键等特征来训练预测模型。

**关键词**: 蛋白质相互作用、机器学习、非共价相互作用、分子特征、预测模型
